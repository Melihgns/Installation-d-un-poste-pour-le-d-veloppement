Installation

Installation de VirtualBox et Ubuntu LTS

Prérequis

VirtualBox

Téléchargez la dernière version depuis le site officiel :

Télécharger VirtualBox

Ubuntu

Téléchargez la version LTS d'Ubuntu Desktop :

Télécharger Ubuntu

Installation VirtualBox

  1. Exécutez le fichier d'installation
  2. Suivez les étapes de l'assistant
  3. Acceptez les composants par défaut
  4. Redémarrez le système si demandé

Installation Ubuntu

  1. Créez une nouvelle machine virtuelle
  2. Sélectionnez l'ISO Ubuntu LTS
  3. Configurez au moins 2GB de RAM
  4. Allouez 20GB d'espace disque minimum
  5. Suivez l'assistant d'installation d'Ubuntu
