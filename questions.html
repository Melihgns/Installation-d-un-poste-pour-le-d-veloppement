Qu'est-ce qu'une machine virtuelle ?

La virtualisation est une technique informatique qui permet de créer une version virtuelle d'une ressource, comme un serveur, un système d'exploitation, une application ou un périphérique de stockage. Elle offre une abstraction des couches matérielles et logicielles pour émuler un autre système. Il existe plusieurs types de virtualisation :

Virtualisation de type 1 (ou virtualisation native, bare metal) :

L'hyperviseur s'exécute directement sur le matériel physique de la machine hôte.

Exemples : VMware ESXi, Microsoft Hyper-V, Stratus.

Virtualisation de type 2 (ou virtualisation hébergée) :

L'hyperviseur s'exécute comme une application sur un système d'exploitation hôte.

Exemples : VMware Workstation, Oracle VirtualBox, Parallels Desktop.

Différences entre émulateur, outil de virtualisation et simulateur

ÉMULATEUR

Imite le matériel d'un système cible sur un système hôte différent. Permet d'exécuter des logiciels conçus pour un autre système ou architecture. Exemples : émulateurs de consoles de jeux (SNES, PlayStation), QEMU.

OUTIL DE VIRTUALISATION

Crée une version virtuelle d'une ressource informatique (serveur, système d'exploitation, stockage, etc.). Permet d'exécuter plusieurs systèmes d'exploitation sur une même machine physique. Exemples : VMware, VirtualBox, Hyper-V.

SIMULATEUR

Modélise le comportement d'un système sans nécessairement reproduire son matériel. Utilisé pour tester, analyser ou prédire le comportement d'un système. Exemples : simulateurs de vol, simulateurs de réseau, simulateurs de circuits électroniques.

Lieu de stockage des fichiers de la machine virtuelle

Les fichiers de la machine virtuelle "Ubuntu 24.04" sont stockés dans le dossier par défaut de VirtualBox : C:\Users\nom_User\VirtualBox VMs\. Le fichier de disque dur virtuel utilise un stockage alloué dynamiquement, ce qui signifie que la taille du fichier .vdi augmentera progressivement jusqu'à la taille maximale définie, au fur et à mesure que l'espace disque sera utilisé dans la machine virtuelle.

Formes des fichiers de la machine virtuelle

Fichier de configuration : extension .vbox
Fichier de disque dur virtuel : extension .vdi (Virtual Disk Image)
