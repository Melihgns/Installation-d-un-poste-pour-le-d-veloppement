Configuration Système

Gestion des utilisateurs et des permissions

Compte Administrateur

Création admin_Gr102_Binome3

Configuration du compte avec privilèges sudo

# Création du compte admin
sudo useradd -m -s /bin/bash admin_Gr102_Binome3

# Attribution des privilèges sudo
sudo usermod -aG sudo admin_Gr102_Binome3

# Définition du mot de passe
sudo passwd admin_Gr102_Binome3

Utilisateurs Standards

# Création des utilisateurs
sudo useradd -m -s /bin/bash BinomeYY_1
sudo useradd -m -s /bin/bash BinomeYY_2

# Création du groupe
sudo groupadd BinomeYY

# Ajout au groupe
sudo usermod -aG BinomeYY BinomeYY_1
sudo usermod -aG BinomeYY BinomeYY_2

Configuration du Prompt

Personnalisation de l'invite de commande avec le nom d'utilisateur :

# Modification du .bashrc
echo 'export PS1="\u@\h:\w\$ "' >> ~/.bashrc

# Rechargement de la configuration
source ~/.bashrc
