Création d'utilisateur et gestion des groupes

Créer l'utilisateur avec la commande sudo adduser --allow-bad-names admin_Gr102_Binome3 : sudo permet d'exécuter la commande avec les droits d'administrateur. adduser est la commande pour ajouter un nouvel utilisateur. --allow-bad-names autorise les noms d'utilisateur contenant des majuscules et des caractères spéciaux, comme c'est le cas ici avec "admin_Gr102_Binome3".

Ajouter l'utilisateur au groupe sudo avec la commande sudo usermod -aG sudo admin_Gr102_Binome3 : usermod permet de modifier les propriétés d'un utilisateur existant. -aG sudo ajoute l'utilisateur au groupe supplémentaire "sudo". L'option -a préserve l'appartenance aux autres groupes.

Vérifier l'appartenance aux groupes avec la commande groups admin_Gr102_Binome3 : Cela affiche "admin_Gr102_Binome3 : admin_Gr102_Binome3 sudo users", confirmant que l'utilisateur appartient maintenant au groupe "sudo" en plus de son propre groupe.

Nous avons reçu comme consigne de créer deux utilisateurs nommés BinomeYY_1 et BinomeYY_2, puis de les associer à un groupe commun nommé BinomeYY. sudo usermod -g binome3 binome3_1 Cette commande modifie le groupe principal de l'utilisateur binome3_1 et l'associe au groupe binome3. De la même manière, cette commande attribue à l'utilisateur binome3_2 le groupe principal binome3, puis on verifie avec groups

Nous devions personnaliser l'invite de commande (prompt) afin qu'elle affiche le nom de l'utilisateur, suivi du symbole @, du nom de la machine, et du chemin d'accès au répertoire courant, accompagné d'un symbole $ pour indiquer un utilisateur standard (ou # pour root). Cependant, ces éléments étaient déjà configurés dans le fichier ~/.bashrc. Nous avons donc choisi de personnaliser davantage le prompt en modifiant la couleur.

Nous avons ouvert le fichier de configuration ~/.bashrc avec la commande suivante : nano ~/.bashrc

Initialement, la couleur était configurée avec le code 01;34 (bleu clair). Nous avons modifié ce paramètre pour utiliser la couleur cyan clair, représentée par le code 01;36.
